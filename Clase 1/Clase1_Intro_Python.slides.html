Entrada de datos¶

Leer datos por teclado en Python se hace usando la funcin input(). Esta funcin genera una pausa en el programa, esperando por una entrada. Dicha entrada, se entiende completada una vez el usuario presiona "enter" para confirmar los datos ingresados.

La funcin input() permite incluir un argumento opcional dentro de los parntesis para entregar un mensaje al usuario.

In [36]:
numero = input("Ingrese un numero: ")

Ingrese un numero: 7

Qu tipo de dato es la variable numero?

Por defecto, todos los valores se guardan en formato string.

In [37]:
numero
Out[37]:
'7'

Guardar siempre los datos de entrada de acuerdo a su tipo ms adecuado. Por ejemplo:

nombre = input('Ingrese su nombre:')
edad = int(input('Ingrese su edad:'))
numero = float(input('Ingrese un numero real:'))

Funciones básicas integradas¶

Python incluye una serie de funciones que estn siempre disponibles y pueden ser utilizadas en cualquier momento.

Por ahora, para entender el concepto, veremos solo la funcin matemtica para redondear un nmero decimal.

Función round()¶

La funcin round() recibe un nmero y de manera opcional un entero que representa la cantidad de decimales para redondear el nmero. Si recibe solo el primer parmetro, aproxima al entero ms cercano.

Sintaxis:

round(nmero, decimales)

La cantidad de decimales es opcional. Si no se especifica, entonces la funcin devolver el entero ms cercano del nmero.

In [44]:
print(round(5.76543, 2))

5.77
In [45]:
print(round(5.76543))

6
